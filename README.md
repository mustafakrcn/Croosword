## 🚀 Özellikler
- İnteraktif bulmaca deneyimi
- Hızlı ve modern arayüz
- Çapraz platform desteği (Capacitor ile mobil uyumlu)

## 🛠️ Kullanılan Teknolojiler
- HTML5 / Vanilla JavaScript / CSS
- [Vite](https://vitejs.dev/) - Frontend geliştirme aracı
- [Capacitor](https://capacitorjs.com/) - Web uygulamasını mobil uygulamaya dönüştürmek için

## 💻 Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için:

1. Proje dizinine gidin ve bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Tarayıcınızda açılan adresten uygulamayı görüntüleyin.

## 📱 Mobil Derleme (Android)

Android platformu için build almak isterseniz:
```bash
npm run build
npx cap sync android
npx cap open android
```
